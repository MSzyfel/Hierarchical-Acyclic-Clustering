\section{Inapproximability Results}
In this section, we show that approximating \textsc{Hierarchical Acyclic Clustering} with uniform edge costs is essentially as hard as approximating \textsc{Hierarchical Clustering}. Since the latter problem cannot be approximated within any constant factor under the Small Set Expansion Hypothesis~\cite{GraphExpansionAndTheUniqueGamesConjecture}, we obtain the same inapproximability result for \textsc{Hierarchical Acyclic Clustering}.

\begin{theorem}[Inapproximability of \textsc{HC} \cite{ApproximateHierarchicalClusteringViaSparsestCutAndSpreadingMetrics}]\label{thm:hc-inapproximability}
    For every $\epsilon > 0$, it is \textsc{SSEH}-hard to distinguish between the following two cases for a given (uniform cost) instance of \textsc{Hierarchical Clustering}:
    \begin{itemize}
        \item \textbf{YES case:} There exists a HC tree $T$ such that $\COST\br{T}\leq \epsilon nm$.
        \item \textbf{NO case:} For every linear arrangement $\pi$ of $V$, $\COST\br{\pi}\geq c\sqrt{\epsilon} nm$, for some constant $c > 0$.
    \end{itemize}
\end{theorem}
Note that since objective of a linear arrangement is a lower bound on the cost of any HC tree, the above result implies that in the \textbf{NO} case, every HC tree $T$ has $\COST\br{T}\geq c\sqrt{\epsilon} nm$.
We now show how to use this result to prove that \textsc{Hierarchical Acyclic Clustering} cannot be approximated within any constant factor under SSEH. 
\begin{theorem}[Inapproximability of \textsc{HAC}]\label{thm:hac-inapproximability}
    For every $\delta > 0$, it is \textsc{SSEH}-hard to distinguish between the following two cases for a given (uniform cost) instance of \textsc{Hierarchical Acyclic Clustering}:
    \begin{itemize}
        \item \textbf{YES case:} There exists a HAC tree $T$ such that $\COST\br{T}\leq \delta nm$.
        \item \textbf{NO case:} Every HAC tree $T$, $\COST\br{T}\geq c'\sqrt{\delta} nm$, for some constant $c' > 0$.
    \end{itemize}
\end{theorem}
\begin{proof}
    Let $G=\br{V,E}$ be an instance of \textsc{Hierarchical Clustering}. We set $\delta = 2\epsilon$ and $c'= c/\sqrt{2}$. We create an \textsc{HAC} instance $H$ as follows: For any vertex $v\in V\br{G}$ we add two new vertices $v_1$ and $v_2$ and we build a triangle on them. 
    
    Assume that we are in the \textbf{YES} case for $G$ and $T$ is the HC tree such that $\COST\br{T}\leq \epsilon nm$. We construct a HAC tree $T'$ for $H$ as follows. We start with $T'=T$. Observe that every leaf cluster in $T'$ is a triangle. Then, we add a new level of the tree, in which we cut each such triangle using one edge. It is easy to see that $\COST\br{T'}\leq 3\COST\br{T}+ 3n\br{G} \leq 3\epsilon n\br{G}m\br{G} + 3n\br{G} \leq \epsilon nm+n\leq \delta nm$, where we used the fact that $\epsilon \geq 1/m$ for large enough $m$. 
    
    Assume that we have a \textbf{NO} case for $G$. We want to show that the cost of any clustering tree $T$ for $H$ has cost at least the cost of any linear arrangement of $G$. Firstly, pick all original vertices of $G$ and project them onto a line according to the order they occur in the leaf clusters of $T$, thus obtaining a linear arrangement of $G$, $\pi$. Consider any two vertices $uv \in E\br{G}$. Assume, that they belong to a different leaf clusters in $T$. Then, we know that the edge $uv$ had to be cut in $T$ and the stretch of $uv$ in $H$ is at least the size of the cluster in which $uv$ was cut. Therefore, assume that $uv$ belongs to the same leaf cluster in $T$. Consider the cluster $C$ in which $uv$ belongs. Let $C'$ be this cluster restricted to the vertices of $G$. Since $C'$ is a tree cluster, we know that the contribution of vertices in $C'$ to the cost of $\pi$ is at most $\spr{C'}^2$. On the other hand, we know that for each vertex $v\in C'$, at least one of the edges $vv_1$ and $vv_2$ has to be cut in $T$ and every such edge contributes to the cost of every vertex in $C'$ so the contribution of vertices in $C'$ to the cost of $T$ is at least $\spr{C'}^2$. By summing up the contributions of all clusters, we obtain that $\COST\br{T}\geq \COST\br{\pi}\geq c\sqrt{\epsilon} nm = c'\sqrt{\delta} nm$.
\end{proof}
