% Abstract
\begin{abstract}
    We consider a variation on the classical \textsc{Hierarchical Clustering} problem, in which we modify the halting condition of the recursive process to stop when the remaining cluster is a tree. We call this problem \textsc{Hierarchical Acyclic Clustering} and we measure the quality of a solution using adapted Dasgupta's clustering objective. This problem can be seen as marriage between \textsc{Feedback Edge Set} and \textsc{Hierarchical Clustering}. By exploiting these connections we obtain an $O\br{\log n \cdot \log\log n}$-approximation algorithm. The main technical contribution of our work is a novel, two-fold way of relaxing the ILP formulation of \textsc{HAC}. Firstly, we use an \textsc{LP} relaxation of the ILP and a simple arithmetic half-up rounding procedure to find the subsets of vertices whose cycles we need to cut at each level of the recursion. After obtaining this information, we use a further \textsc{SDP} relaxation in order to find the subsets of edges we need to cut in order to decrease the size of the instance efficiently. The key ingredient in our algorithm is that both LP and SDP solutions can be decomposed into several instances of the relaxations of \textsc{Feedback Edge Set} and \textsc{$k$-Balanced Partitioning} respectively. This allows us to use already known rounding procedures for the aforementioned problems as essential subroutines in our algorithm. To complement this result, we show that the problem cannot be approximated within any constant factor under the \textsc{Small Set Expansion Hypothesis}.
\end{abstract}
