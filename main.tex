% Abstract
\begin{abstract}
    We consider a variation on the classical \textsc{Hierarchical Clustering} problem, in which we modify the halting condition of the recursive process to stop when the remaining cluster is a tree. We call this problem \textsc{Hierarchical Acyclic Clustering} and we measure the quality of a solution using adapted Dasgupta's clustering objective. This problem can be seen as marriage between \textsc{Feedback Edge Set} and \textsc{Hierarchical Clustering}. By exploiting these connections we obtain an $O\br{\log n\log\log n}$-approximation algorithm. The main technical contribution of our work is a novel, two-fold way of relaxing the ILP formulation of \textsc{HAC} in order to extract valuable information in polynomial time. Firstly, we use an \textsc{LP} relaxation of the ILP to gain information about the vertices whose cycles we need to cut at each level of the recursion. Secondly, we use a further \textsc{SDP} relaxation in order to gain information about the edges we need to cut in order to decrease the size of the instance efficiently. The key ingredient in our algorithm is that both LP and SDP solutions can be decomposed into several solutions to the relaxations of the \textsc{FES} and \textsc{$k$-Balanced Partitioning} instances respectively. This allows us to use already known rounding procedures for the aforementioned problems as essential subroutines in our procedure. To complement this result, we show that the problem cannot be approximated within any constant factor under the Small Set Expansion Hypothesis.
\end{abstract}
