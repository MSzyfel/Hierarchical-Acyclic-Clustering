% Abstract
\begin{abstract}
    We consider a variation on the classical \textsc{Hierarchical Clustering} problem, in which we modify the halting condition of the recursive process to stop when the remaining cluster is graph belonging to a given class $\cF$. We call this problem \textsc{Hierarchical Clustering into $\cF$-clusters} and we measure the quality of a solution using adapted Dasgupta's clustering objective. We study two natural choices of $\cF$: trees $\cT$ and bounded diameter graphs $\cD$ (which includes cliques). We present polynomial time $O\br{\log n\log\log n}$-approximation for clustering into $\cT$-custers and $O\br{\log n}$-approximation for clustering into $\cD$-clusters. 
    The main technical contribution of our work is a novel, two-fold way of relaxing the ILP formulation of \textsc{HC-$\cF$} for both $\cT$ and $\cD$. Firstly, we use an \textsc{LP} relaxation of the ILP and a simple arithmetic half-up rounding procedure to find the subsets of vertices which need to belong to a $\cF$-cluster at each level of the recursion. After obtaining this information, we use a further \textsc{SDP} relaxation in order to find the subsets of edges we need to cut in order to decrease the size of the instance efficiently. The key ingredient in our algorithm is that both LP and SDP solutions can be decomposed into several instances of the relaxations of \textsc{Feedback Edge Set}/\textsc{Min-Multicut} and \textsc{$k$-Balanced Partitioning} respectively. This allows us to use already known rounding procedures for the aforementioned problems as essential subroutines in our algorithm. To complement this result, we show that for both $\cT$ and $\cD$ the problem cannot be approximated within any constant factor under the \textsc{Small Set Expansion Hypothesis}.
\end{abstract}
